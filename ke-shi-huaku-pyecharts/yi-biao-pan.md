# 1.Gauge\(仪表盘\)

> Gauge.add\(\) 方法：

```
add(name, attr, value,
    scale_range=None,
    angle_range=None, **kwargs)
```

* name -&gt; str   图例名称

* attr -&gt; list   属性名称

* value -&gt; list   属性所对应的值

* scale\_range -&gt; list   仪表盘数据范围。默认为 \[0, 100\]

* angle\_range -&gt; list   仪表盘角度范围。默认为 \[225, -45\]
